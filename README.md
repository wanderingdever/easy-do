# 开户

> 以下只包含连连支付的接口, 还未涉及到我们自己的数据处理功能。用户体系，操作记录，文件管理，账户数据

## 个人用户

1. 绑定手机验证码申请
2. 绑定手机验证码验证
3. 个人开户申请
4. 随机密码因子获取
5. 个人开户验证
6. 个人开户异步通知

## 企业用户

1. 文件上传（建议）
2. 绑定手机验证码申请
3. 绑定手机验证码验证
4. 随机密码因子获取
5. 企业开户申请（上传相应证件照片）
6. 企业开户验证（个体工商户开户需要）
7. 企业开户异步通知

# 充值/消费

## 充值

### 银行卡快捷支付

1. 统一支付创单
2. 随机密码因子获取
3. 银行卡快捷支付
4. 交易二次短信验证
5. 支付结果异步通知

### 网关类支付（网银）

1. 统一支付创单
2. 随机密码因子获取(组合余额支付时需要)
3. 网关类支付
4. 交易二次短信验证(组合余额支付时需要)
5. form表单POST请求网银收银台地址gateway_url
6. 支付结果异步通知

### 网关类支付（支付宝/微信）

1. 统一支付创单
2. 随机密码因子获取(组合余额支付时需要)
3. 网关类支付
4. 交易二次短信验证(组合余额支付时需要)
5. 参考官方文档用返回的payload里的参数请求官方支付
6. 支付结果异步通知

### 委托代扣(网关)

1. 委托代扣
2. 支付结果异步通知

# 提现

## 用户提现

1. 随机密码因子获取
2. 提现申请
3. 交易二次短信验证（默认需要）
4. 提现确认（提现申请check_flag为Y时需要）
5. 提现结果异步通知

## 平台提现

1. 提现申请
2. 提现确认（提现申请check_flag为Y时需要）
3. 提现结果异步通知

# 代发

## 用户代发

1. 随机密码因子获取接口
2. 代发申请接口
3. 交易二次短信验证（默认需要）
4. 代发结果异步通知交易

## 平台代发

1. 代发申请接口
2. 代发结果异步通知交易
